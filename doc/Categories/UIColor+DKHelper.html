blueColorWithAlpha:

Creates and returns a blue color object using the specified opacity.

+ (UIColor *)blueColorWithAlpha:(CGFloat)alpha

Parameters

alpha

The opacity value of the color object, specified as a value from 0.0 to 1.0.

Return Value

A blue UIColor object.

Discussion

Creates and returns a blue color object using the specified opacity.

Values below 0.0 are interpreted as 0.0, and values above 1.0 are interpreted as 1.0.

The color information represented by this object is in the device RGBA colorspace.
