currentDayDate

Returns a new NSDate object containing the current date of the day with a IOS8601 default format. The number of hours, minutes, seconds and the GMT timezone is set to 0.

+ (NSDate *)currentDayDate

Return Value

A NSDate containing the current date of the day. The date format is the ISO8601 one.

Discussion

dateFromString:format:

Returns a new NSDate object created from a string with a specified format.

+ (NSDate *)dateFromString:(NSString *)string format:(NSString *)format

Parameters

string

The NSString object representing the date.

format

The NSString object representing the format in which the string is formatted.

Return Value

A NSDate created from a given string. NULL if the date isn’t valid.

Discussion

Returns a new NSDate object created from a string with a specified format.

Example of code will be [NSDate dateFromString:@“1970-01-01” format:@“yyyy-MM-dd”];
