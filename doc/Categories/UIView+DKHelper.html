roundCorner:radius:

Rounds the specified corner(s) of the receiver with a given radius.

- (void)roundCorner:(UIRectCorner)corner radius:(CGFloat)cornerRadius

Parameters

A bitmask value that identifies the corners that you want rounded. You can use this parameter to round only a subset of the corners of the rectangle.
